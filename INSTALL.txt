=====================================
NUSA BITES - PANDUAN INSTALASI
=====================================

Terima kasih telah menggunakan Nusa Bites!
Ikuti langkah-langkah berikut untuk instalasi:

=====================================
1. PERSYARATAN SISTEM
=====================================

- PHP 7.4 atau lebih tinggi
- MySQL 5.7 atau lebih tinggi  
- Apache Web Server (dengan mod_rewrite)
- XAMPP/WAMP/LAMP (recommended untuk Windows/Mac/Linux)

=====================================
2. LANGKAH INSTALASI
=====================================

STEP 1: PERSIAPAN FILE
-----------------------
1. Extract file project ke folder htdocs di XAMPP
   Contoh: C:\xampp\htdocs\NusaBites

STEP 2: BUAT DATABASE
----------------------
1. Jalankan XAMPP Control Panel
2. Start Apache dan MySQL
3. Buka browser, akses: http://localhost/phpmyadmin
4. Klik tab "SQL"
5. Buka file "database.sql" dengan text editor
6. Copy semua isinya
7. Paste di SQL editor phpMyAdmin
8. Klik tombol "Go" atau "Kirim"
9. Database "nusabites" akan terbuat beserta data sample

STEP 3: KONFIGURASI DATABASE (Opsional)
----------------------------------------
Jika password MySQL Anda bukan kosong:

1. Buka file: config/database.php
2. Edit baris berikut sesuai setting MySQL Anda:
   
   define('DB_HOST', 'localhost');
   define('DB_USER', 'root');
   define('DB_PASS', '');          // Isi password MySQL
   define('DB_NAME', 'nusabites');

3. Save file

STEP 4: JALANKAN APLIKASI
--------------------------
1. Pastikan Apache dan MySQL di XAMPP sudah running
2. Buka browser
3. Akses: http://localhost/NusaBites
4. Website sudah bisa digunakan!

=====================================
4. TROUBLESHOOTING
=====================================

MASALAH: "Cannot connect to database"
SOLUSI:
- Pastikan MySQL service running di XAMPP
- Cek username/password di config/database.php
- Pastikan database "nusabites" sudah dibuat

MASALAH: CSS tidak muncul
SOLUSI:
- Clear cache browser (Ctrl+F5)
- Pastikan folder assets/css/style.css ada
- Cek permission folder (chmod 755)

MASALAH: Error 404 Not Found
SOLUSI:
- Pastikan mod_rewrite Apache aktif
- Cek file .htaccess ada di root folder
- Restart Apache di XAMPP

MASALAH: Session tidak berfungsi
SOLUSI:
- Pastikan folder tmp di PHP bisa ditulis
- Cek php.ini, pastikan session.save_path valid
- Restart Apache

MASALAH: Upload gambar tidak berfungsi
SOLUSI:
- Saat ini sistem menggunakan URL gambar eksternal
- Untuk upload lokal, perlu modifikasi code
